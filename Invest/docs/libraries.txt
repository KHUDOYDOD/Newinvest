
ОСНОВНЫЕ БИБЛИОТЕКИ

React и TypeScript:
- react: Основная библиотека для создания пользовательского интерфейса
- react-dom: Рендеринг React компонентов
- typescript: Язык программирования с поддержкой типизации

UI Компоненты и Стили:
- tailwindcss: CSS фреймворк для быстрой стилизации
- @radix-ui/*: Доступные UI компоненты
- class-variance-authority: Создание вариантов компонентов
- tailwind-merge: Объединение Tailwind классов
- lucide-react: Набор иконок
- date-fns: Работа с датами
- recharts: Создание графиков и диаграмм
- framer-motion: Анимации

Навигация:
- wouter: Легкий роутер для React

Формы и Валидация:
- react-hook-form: Управление формами
- zod: Валидация данных
- @hookform/resolvers: Интеграция валидаторов

Серверная часть:
- express: Веб-фреймворк для Node.js
- @neondatabase/serverless: Работа с базой данных
- drizzle-orm: ORM для TypeScript
- passport: Аутентификация
- express-session: Управление сессиями

Запросы и Кэширование:
- @tanstack/react-query: Управление серверным состоянием

Разработка:
- vite: Инструмент сборки
- esbuild: Быстрый бандлер JavaScript
- tsx: Запуск TypeScript файлов
- @vitejs/plugin-react: Интеграция React с Vite

Дополнительные утилиты:
- clsx: Условное объединение классов
- cmdk: Создание командной палитры
- ws: WebSocket клиент и сервер
- zod-validation-error: Улучшенные сообщения об ошибках валидации

DevDependencies:
- autoprefixer: Автопрефиксер CSS
- postcss: Инструмент для трансформации CSS
- @types/*: TypeScript типы для библиотек
- drizzle-kit: Инструменты для работы с базой данных
